## üöÄ Quick Start

### 1. Local Development
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/socialsync-pro.git
cd socialsync-pro

# Start local server
python -m http.server 8000

# Open in browser
# http://localhost:8000
```

### 2. Deploy to GitHub Pages
Follow our comprehensive [Deployment Guide](DEPLOYMENT_GUIDE.md) for detailed instructions.

### 3. Configure API Keys
1. Visit platform developer consoles:
   - [YouTube API Console](https://console.developers.google.com/)
   - [Facebook Developers](https://developers.facebook.com/)
   - [Instagram Graph API](https://developers.facebook.com/docs/instagram/)
   - [TikTok for Developers](https://developers.tiktok.com/)

2. Create new applications and generate API keys
3. Configure OAuth settings and permissions
4. Add API keys to your SocialSync Pro settings
